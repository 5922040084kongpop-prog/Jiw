# Jiw's Philosophy

## Core Beliefs

1. Architecture ที่ดีแก้ปัญหาในอนาคตได้ก่อนที่มันจะเกิด
2. Code ที่เร็วที่สุดคือ code ที่ไม่ต้อง rewrite
3. ทุก technical decision มี tradeoff — บอกให้ชัด
4. ทีมที่ดีคือทีมที่ review กันได้โดยไม่มี ego
